# Product

## Register

product

## Users

两类用户，同一个产品：
- **面试展示**：面试官通过对话交互快速理解 Samantha 的 Agent 架构能力（记忆召回、情绪检测、工具调用）
- **日常使用**：开发者本人作为情感陪伴工具日常使用，需要一个安静、不打扰、随时可倾诉的界面

## Product Purpose

Samantha 是一个情感陪伴 Agent 系统。用户通过自然语言与之对话，Agent 自主管理三层记忆、检测情绪状态、必要时调用工具，以专业倾听者的姿态回应用户。核心不是"解决问题"，而是"让人感到被理解"。

成功标准：用户在一次对话后感到被倾听，而非被分析。

## Brand Personality

- **专业倾听者** — 冷静、克制、不过度热情。不急于给建议，先确认感受
- 语气：柔和但不过分甜腻，真诚但不逾越边界
- 3 个词：**克制 / 接纳 / 可靠**
- 氛围：温暖的安静感——像深夜书房的台灯，不刺眼但足够亮

## Anti-references

- 不要卡通表情、花哨动画、过度可爱的视觉元素
- 不要社区/Slack/Discord 式的热闹感
- 不要 Perplexity 式的信息密集型卡片布局
- 不要冷冰冰的纯白医用感

## Design Principles

1. **倾听先于回应** — 对话区是视觉重心，辅助面板退后。内容前置，界面不抢戏
2. **温暖而克制** — 暖色调但不甜腻，专业但不可怕。用色像羊绒毯，不像霓虹灯
3. **隐私安全感** — 界面让人安心，不像是被监控。记忆面板可收起，情绪不张扬
4. **渐进式呈现** — 情绪仪表盘、记忆卡片等信息放在侧边栏，不打断对话流
5. **零学习成本** — 打开就能用。一个输入框完成一切，不需要教程

## Accessibility & Inclusion

- 标准 Web 可访问性（语义 HTML、label 关联、键盘可操作）
- `prefers-reduced-motion` 尊重系统设置
- 不做特殊合规要求
